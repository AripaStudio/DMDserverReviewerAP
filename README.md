## A simple C# watchdog utility designed to fix a memory leak issue in **Visual D**. It monitors the `dmdserver` process and automatically kills it if its memory usage exceeds a specified threshold. Once terminated, Visual D instantly restarts a fresh, clean instance of the server.

This software is very simple, and all its code is contained in this file:
- [open](https://github.com/AripaStudio/DMDserverReviewerAP/blob/master/DMDserverReviewerAP/Program.cs)

**This project is based on .NET 9.**
